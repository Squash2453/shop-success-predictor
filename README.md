# 🏪 Shop Success Predictor

## 📌 Overview

This project is a Machine Learning-based recommendation system that predicts the **top 3 most suitable shop types** for a given location along with their **success probability (%)**.

It helps in making **data-driven business decisions** by analyzing factors such as location, demand, and other relevant features.

---

## 🚀 Features

* Predicts **top 3 shop types**
* Provides **success probability (%)**
* Built using **Random Forest**
* Interactive UI using **Streamlit**
* Easy to use and scalable

---

## 🧠 Problem Statement

Choosing the right type of shop in a specific location is challenging and risky.

This project aims to:

* Analyze input features (like location/business factors)
* Predict the most profitable shop types
* Reduce business failure risk

---

## ⚙️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn (Random Forest)
* Streamlit (for UI)

---

## 📊 Dataset

The dataset used in this project is available on Kaggle:

🔗 https://www.kaggle.com/datasets/shiblink/shop-success-rate/data

The dataset includes features related to shop performance, location factors, and business metrics used for training the model.

---

## 📊 How It Works

1. User provides input data (location/business features)
2. The trained ML model processes the input
3. Model predicts probabilities for each shop type
4. Top 3 shop types are displayed based on highest probability
